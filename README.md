A fun platformer game for kids featuring a wizard navigating through a magical mountain!

## How to Play

- **Arrow Keys or A/D**: Move left and right
- **Spacebar**: Jump
- **Z or Mouse Click**: Shoot fireballs
- **ESC**: Pause game

## Game Objectives

1. Navigate through the mountain levels
2. Collect crystals to gain points and restore health
3. Destroy obstacles (boulders, crates, stones) with your fireballs
4. Avoid falling into pits
5. Reach the end of the level to complete the adventure!

## Game Features

- Magical wizard character with smooth animations
- Colorful mountain environment with parallax scrolling
- Various obstacles: boulders, wooden crates, stone blocks
- Collectible crystals for scoring
- Health system with damage and healing
- Progressive difficulty levels
- Score tracking and completion screen

## Getting Started

Simply open `index.html` in any modern web browser and start playing!
